Title: krkr console plugin
Author: わたなべごう

●これはなに？

吉里吉里Zの起動コンソール(DOS窓)に入力プロンプトを追加するプラグインです
※そのままだとセキュリティ的に問題がありますので開発用にのみ使うようにしてください

●使い方

プラグインをロードすると、コンソールからの入力をTJS式として実行するようになります
るようになります。結果もそのままコンソールに表示されます

●ライセンス

このプラグイン自身のライセンスは吉里吉里Z のライセンスと同一として扱って下さい。

wineeditline (BSDライセンス) を使っています

https://github.com/winlibs/wineditline
