Title: krkr console plugin
Author: わたなべごう

●これはなに？

吉里吉里Zのコンソールに入力プロンプトを追加するプラグインです
セキュリティ的に問題がありますので開発用にのみ使うようにしてください

●使い方

プラグインをロードすると、コンソールが存在する場合にそこからの入力
をTJS式として実行するようになります。結果もそのままコンソールに表示されます

●ライセンス

このプラグイン自身のライセンスは吉里吉里Z のライセンスと同一として扱って下さい。

https://github.com/winlibs/wineditline
